# Настройка DNS для Resend — globalfashioncode.com

Инструкция для владельца домена. Нужно добавить **3 DNS-записи**, чтобы письма от сайта (заявки с формы) уходили с адреса `@globalfashioncode.com` и нормально доходили до получателей.

---

## Куда заходить

Личный кабинет регистратора, где куплен `globalfashioncode.com` → раздел **DNS / Управление зоной / DNS-записи**. Там должна быть кнопка вроде «Добавить запись».

---

## Запись 1 — DKIM (TXT)

| Поле | Значение |
|---|---|
| Type / Тип | **TXT** |
| Name / Host / Имя | `resend._domainkey` |
| Content / Value / Значение | скопировать из Resend полностью (строка начинается с `p=MIGfMA…` и заканчивается `…QIDAQAB`, она длинная ~400 символов) |
| TTL | Auto / по умолчанию |

---

## Запись 2 — SPF (MX)

| Поле | Значение |
|---|---|
| Type / Тип | **MX** |
| Name / Host / Имя | `send` |
| Content / Value / Значение | скопировать из Resend (вид: `feedback-smtp.<регион>.amazonses.com`) |
| Priority / Приоритет | **10** |
| TTL | Auto |

---

## Запись 3 — SPF (TXT)

| Поле | Значение |
|---|---|
| Type / Тип | **TXT** |
| Name / Host / Имя | `send` |
| Content / Value / Значение | скопировать из Resend (вид: `v=spf1 include:amazonses.com ~all`) |
| TTL | Auto |

---

## Важные нюансы

1. **В поле Name писать только префикс**, без `.globalfashioncode.com` в конце.
   - Правильно: `send`
   - Неправильно: `send.globalfashioncode.com`
   - У некоторых регистраторов наоборот требуется полное имя с точкой на конце (`send.globalfashioncode.com.`). Если непонятно — смотрите подсказки в интерфейсе.

2. **Content копировать целиком.** В Resend длинные значения сокращены в виде превью (`p=MIGfMA[…]QIDAQAB`). Нужно навести курсор на значение и нажать кнопку **Copy** — скопируется полная строка.

3. **Кавычки руками не добавлять.** Если регистратор сам оборачивает TXT-значение в кавычки — это нормально. Сами не вписывайте.

4. **Если на корневом домене уже есть SPF-запись** (TXT с `v=spf1 …`) — её не трогать. Новая SPF-запись у нас на поддомене `send`, она отдельная и не конфликтует.

5. После сохранения подождать **10–60 минут**, затем в Resend нажать **Verify DNS Records**. Если все три статуса зелёные — готово.

---

## Если что-то непонятно

Пришлите скриншот интерфейса регистратора (страница «Добавить DNS-запись») — покажем прямо на нём, куда вписывать.
